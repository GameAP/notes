Notes v1.0 for GameAP
Модуль заметок. Позволяет пользователям создавать заметки в панели.


----------- Установка -------------

1. Скопируйте директорию application из архива в корневую директорию с панелью на сервере.

2. Обновите список модулей. Для этого зайдите в панель и перейдите 
Модули -> Управление модулями -> Обновить список модулей

3. Можете добавить ссылку на модуль в боковом меню. Боковое меню редактируется в файле
application/views/default/profile.html
